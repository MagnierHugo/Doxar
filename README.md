### DOXAR

Doxar is game engine made during a 2 month long School project at G-Tech (Gaming Campus)

### Features

- Assets loading
- 3D model rendering
- Intuitive UI
- Files browsing
- Scene building
- Scripting (Unfinished)
- lighting (Unfinished)
- Physics (Unfinished)
